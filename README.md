## How to create a PR?

1. Fork this repo
2. Get on the version of this repo present on your account
3. Clone it
4. Create a new branch (don't work on main branch and follow the convention for naming the branch "username/type of work/the work" for instance singharyan006/feat/logic)
5. Create a new file (any file)
6. Fill content or leave it empty your wish
7. then add it > commit it > push it
8. You would be able to see a "Compare and Pull Request" click on it
9. Give a suitable title to the PR
10. Give the description about the work you did
11. Click on Create the PR.
